# my_api

Проект - веб-приложение на Fast API, которое позволяет предсказывать вероятность увольнения сотрудников на основе ml-классификатора Catboost. 

Входные данные: уровень стресса ("Stress Level") и удовлетворенность работой ("Job Satisfaction"), оба значения являются числами от 1 до 10 включительно.  
Выходные данные: 1 в случае увольнения и 0 в противоположном.

Для запуска приложения необходимо скачать репозиторий и запустить файл "app.py".
