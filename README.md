# my_api

Проект представляет собой веб-приложение на Fast API, которое позволяет предсказывать вероятность увольнения сотрудников на основе модели машинного обучения Catboost. 

Модель принимает два параметра: уровень стресса (stress_level) и удовлетворенность работой (job_satisfaction), оба значения являются числами от 1 до 10 включительно.  
Результатом является 1 в случае увольнения и 0 в противоположном.

Для запуска приложения необходимо скачать репозиторий и запустить файл "app.py"
